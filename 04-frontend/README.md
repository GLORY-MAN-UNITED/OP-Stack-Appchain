# 04. Frontend

Build the wallet connection, transfer UI, and on-chain data views.

This folder is the direct work area for the payment UI and wallet integration.

## Responsibilities

- Create wallet connection screens
- Build the transfer page
- Fetch and render transaction history
- Show merchant receipt dashboards

## Stack

- Next.js or React
- Tailwind CSS, Shadcn UI, or MUI
- Wagmi + Viem
- RainbowKit
