# 02. Smart Contract

Build the on-chain payment logic with a focus on safety and simplicity.

This folder is the direct work area for contracts, tests, and deployment scripts.

## Responsibilities

- Implement a mock ERC-20 for test funds
- Build the core payment contracts
- Add the fee model for small, high-frequency payments
- Handle basic authorization and access control

## Stack

- Solidity
- Foundry or Hardhat
- OpenZeppelin Contracts
