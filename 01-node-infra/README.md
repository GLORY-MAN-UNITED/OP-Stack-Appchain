# 01. Node & Infra

Deploy the OP Stack testnet, configure Sequencer and RPC nodes, and keep the L1-L2 bridge stable.

- Deploy the OP Stack testnet locally or on a cloud server

- Debug cross-chain and bridge flows between Sepolia and the appchain
- Maintain automation scripts for deployment and ops

## Stack

- OP Stack monorepo
- Golang
- Shell scripting
- Docker, Docker Compose, Linux
