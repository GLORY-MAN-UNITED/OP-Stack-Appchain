# MVP Plan

Target demo:

- Local OP Stack deployment on one machine
- Sepolia as the L1 testnet
- One mock ERC-20
- One payment contract
- One frontend for wallet connect, transfer, and merchant records
- One load test script
- One comparison table against Sepolia transfers

Simplest deployment choice:

- Do not use a server first.
- Use local deployment for the appchain.
- Use Sepolia only for L1 and benchmark comparison.
