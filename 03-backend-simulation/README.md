# 03. Backend & Simulation

Generate high-concurrency traffic and collect performance data such as TPS, latency, and gas usage.

## Responsibilities

- Write load-testing scripts
- Simulate large volumes of user and merchant transactions
- Collect execution time, gas usage, and block confirmation time
- Produce reproducible benchmark data

## Stack

- Golang + go-ethereum
- Or Node.js + Ethers.js
- Prometheus + Grafana
